# Talent Academy – Data Analysis & Preprocessing

Bu projede sağlık verisi içeren bir dataset üzerinde
**Keşifsel Veri Analizi (EDA)** ve **veri ön işleme** adımları uygulanmıştır.

## Kullanılan Teknolojiler
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Yapılan İşlemler
- Veri setinin incelenmesi (shape, info, describe)
- Eksik değer analizi
- Histogram, scatter plot, countplot
- Korelasyon analizi (heatmap)
- Eksik verilerin doldurulması
- Sayısal verilerin standardizasyonu
- Kategorik verilerin One-Hot Encoding ile dönüştürülmesi
